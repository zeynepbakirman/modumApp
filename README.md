modumApp - Öğrenci Esenlik ve Destek Sistemi 🎓

modumApp, Samsun Üniversitesi öğrencilerinin akademik stres, kişisel esenlik ve kampüs güvenliği süreçlerinde yanlarında olan; anonimlik, uzman rehberliği ve yerel destekleri bir araya getiren kapsamlı bir mobil uygulamadır.

Uygulamanın tüm özelliklerini, 9 haneli öğrenci no maskeleme mantığını ve çalışma performansını aşağıdaki YouTube bağlantısından izleyebilirsiniz:

✨ Öne Çıkan Özellikler

Mood Tracker (Duygu Takibi): Kullanıcıların günlük modlarını kaydederek, ruh hallerine göre Samsun sahil yürüyüşü veya kampüs içi aktiviteler gibi yerel tavsiyeler almasını sağlar.

Anonimlik ve Güvenlik: Öğrenci numaralarının son 3 hanesi maskelenerek veri gizliliği sağlanır, güvenli bir kullanıcı deneyimi sunulur.

Güvenlik Paneli & KADES: Samsun Üniversitesi Canik Kampüsü koordinatları, acil durum butonları ve KADES entegrasyonu ile öğrenci güvenliğini önceler.

Esenlik Rehberi & YouTube API: Uyku, meditasyon ve nefes egzersizleri gibi içeriklere entegre YouTube oynatıcısı üzerinden doğrudan erişim.

İnteraktif Anketler ve Fiziksel Takip: Boy, kilo ve sağlık verilerinin takibi ile kişiselleştirilmiş "Yol Haritası" (Roadmap) ekranı.

🛠 Teknik Detaylar & Mimari
Framework: React Native + Expo (SDK 54)

Dil: TypeScript

Veri Yönetimi: Context API (UserContext) ve AsyncStorage ile kalıcı/dinamik veri akışı.

Tasarım: Modern Linear Gradient geçişleri, özelleştirilmiş bileşenler ve Lucide-React ikon seti.

Navigasyon: Expo Router ile dosya tabanlı yönlendirme yapısı.

⚙️ Kurulum ve Çalıştırma
Projeyi yerel ortamınızda çalıştırmak için:

Repoyu klonlayın: git clone <repo-url>

Proje dizinine (frontend) gidin: cd frontend

Bağımlılıkları yükleyin: npm install

Uygulamayı başlatın: npx expo start -c
